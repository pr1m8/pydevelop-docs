class core.data_processor.DataProcessor(config: Dict[str, Any] | None = None, validate: bool = True)[source]

Core data processor for the Haive ecosystem.

This class provides the fundamental data processing capabilities that are used throughout the Haive ecosystem. It handles data validation, transformation, and error management.

Parameters:

  • config – Configuration dictionary for the processor

  • validate – Whether to validate input data (default: True)

Examples

Basic usage:

>>> processor = DataProcessor()
>>> data = {"name": "test", "value": 42}
>>> result = processor.process(data)
>>> print(result.status)
'success'

With custom configuration:

>>> config = {"strict_mode": True, "timeout": 30}
>>> processor = DataProcessor(config=config)
>>> result = processor.process(data)
