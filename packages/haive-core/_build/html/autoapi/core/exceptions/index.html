Bases: Exception

Base exception for all Haive-related errors.

This is the root exception class for the Haive ecosystem. All other Haive exceptions should inherit from this class.

message[source]

Error message

error_code[source]

Optional error code for categorization

context[source]

Optional context dictionary with additional error details

Examples

Basic usage:

>>> raise HaiveError("Something went wrong")

With error code and context:

>>> raise HaiveError(
...     "Validation failed",
...     error_code="VALIDATION_ERROR",
...     context={"field": "email", "value": "invalid"}
... )
