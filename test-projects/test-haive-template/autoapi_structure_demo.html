🎯 The Solution

Our AutoAPI Hierarchical Organization Analysis document provides the complete fix:

  1. Configuration Change: Set autoapi_own_page_level = "module"
  2. Custom Templates: Modify AutoAPI templates for package-based organization
  3. Furo Theme Config: Set navigation_depth = 4 and collapse_navigation = False
  4. Template Customization: Override python/index.rst to group by package
